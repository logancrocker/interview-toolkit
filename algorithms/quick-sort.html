Quick sort


Quick sort, like Merge sort, uses the divide and conquer strategy and is one of the most popular sorting algorithms along with Radix sort. It has a worst case time complexity of O(n2) and a best case of O(nlogn).

Implementation


Quick sort works by picking an element as a pivot and then partitioning the array around the pivot. The partition() function is where most of the work happens. The partition function is supposed to place the partition at its correct position and then put every item smaller than it before it and every item bigger than it after it. This is done in linear time.

Examples


The following implementation uses the right-most element as the pivot, but there are multiple ways of selecting the pivot. A few other examples are always using the first element, using a random element, and using the median. 


// C++
void swap() {
    int t = *a; 
    *a = *b; 
    *b = t; 
} 

int partition(int arr[], int left, int right) {
  int pivot = arr[right];
  int i = left - 1;

  for (int j = left; j <= right - 1; ++j) {
    if (arr[j] <= pivot) {
      i++;
      swap(&arr[i], &arr[j]);
    }
  }
  swap(&arr[i + 1], &arr[right]);
  return (i + 1);
}

void quickSort(int arr[], int left, int right) {
  if (left < right) {
    int partitionIndex = partition(arr, left, right);

    quickSort(arr, left, partitionIndex - 1);
    quickSort(arr, partitionIndex + 1, right);
  }
}
            

# Python
def partition(arr, left, right):
  i = left - 1
  pivot = arr[right]

  for j in range(left, right):

    if arr[j] <= pivot:
      
      i = i + 1
      arr[i], arr[j] = arr[j], arr[i]

  arr[i + 1], arr[right] = arr[right], arr[i + 1]

  return i + 1

def quickSort(arr, left, right):
  
  if left < right:

    partitionIndex = partition(arr, left, right)

    quickSort(arr, left, partitionIndex - 1)
    quickSort(arr, partitionIndex + 1, right)

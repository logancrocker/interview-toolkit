Binary Search


Binary search is an algorithm used for searching linear data structures which are already sorted. It is typically the best option for this situation, since it has a time complexity of O(logn).

Implementation


Binary Search works by repeatedly slicing the data in half. It picks the value in the middle, and if the value we are looking for is greater than that, it ignores the lower half and only looks at the upper half of the data. It continues doing this, reducing the data by half each time until either the middle value is what we want or it doesn't find the value. This algorithm lends itself to recursion, but you can also implement it iteratively — which is true for any recursive algorithm.

Examples


The following implementations will return the index of the value if it is found, or -1 otherwise.

Recursive 


// C++
int recursive(int arr[], int left, int right, int value) {
  if (right >= left) {
    int mid = left + (right - left) / 2;

    if(arr[mid] == value) {
      return mid;
    }
    else if (arr[mid] > value) {
      return recursive(arr, left, mid - 1, value);
    }
    else {
      return recursive(arr, mid + 1, right, value);
    }
  }
  return -1;
}
            

# Python
def recursive(arr, left, right, value):
  if (left <= right):
    mid = left + (right - left) / 2

    if (arr[mid] == value):
      return mid

    elif (value < arr[mid]):
      return recursive(arr, left, mid - 1, value)

    else:
      return recursive(arr, mid + 1, right, value)
  
  return -1

Iterative 


// C++
int iterative(int arr[], int left, int right, int value) {
  while (left <= right) {
    int mid = left + (right - left) / 2;

    if (arr[mid] == value) {
      return mid;
    }
    else if (arr[mid] < value) {
      left = mid + 1;
    }
    else {
      right = mid - 1;
    }
  }
  return -1;
}
            

# Python
def iterative(arr, left, right, value):
  while (left <= right):
    mid = left + (right - left) / 2

    if (arr[mid] == value):
      return mid

    elif (value < arr[mid]):
      right = mid - 1

    else:
      left = mid + 1

  return -1

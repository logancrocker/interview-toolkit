Linked Lists


You've probably heard of arrays, or lists if you use Python. Well a linked list is similar to these data structures, except the data within a Linked List is not stored contiguously in memory. This is done by using pointers.
There are two implementations of Linked Lists: doubley-linked and singley-linked. In a doubley-linked list, each node has a pointer to its next and previous nodes. In a singley-linked list, each node only has a pointer to its next node. This means you can traverse from head to tail for both, but can only traverse from tail to head for a doubley-linked list.

Implementation


A linked list is implemented with a Node class, which contains a pointer to next and a variable containing that node's data. If you are implementing a doubley-linked list, there will also be a pointer to prev.
You can also create a LinkedList wrapper class which only contains the head Node.

Examples


Below is an implementation of a singley linked list.


// C++
class Node {
  public:
    Node* next;
    int data; 
}
class LinkedList {
  public:
    Node* head;
};
                

# Python
class Node:
  def __init__(self, data):
    self.data = data
    self.next = None

class LinkedList:
  def __init__(self, head):
    self.head = head
